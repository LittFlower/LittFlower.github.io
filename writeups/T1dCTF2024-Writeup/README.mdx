<p>[easy] Repeater</p>
<p>As is well known, CTFers are all repeaters!</p>
</blockquote>
<p>栈溢出题目，<code class="inline-code">sub_11C9()</code> 可以栈溢出，只修改返回地址的 1 字节，可以做到 partial overwrite 无限循环 <code class="inline-code">sub_11C9()</code> 读入 payload。</p>
