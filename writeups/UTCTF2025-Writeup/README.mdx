<p>可以看到给了 JSArray 加了一个 <code class="inline-code">confuse</code> 方法，实现 <code class="inline-code">PACKED_DOUBLE_ELEMENTS</code> 和 <code class="inline-code">PACKED_ELEMENTS</code> 相互转化，复习一下知识点：</p>
<ul>

</ul>
