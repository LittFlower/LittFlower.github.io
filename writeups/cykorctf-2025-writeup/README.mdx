<h2 id="shortcut" class="scroll-target"><a href="#shortcut"><code class="hash inline-code">#</code>shortcut</a></h2>
<p>牢了最久的题。。</p>
<ul>
<li><code class="inline-code">add</code> <code class="inline-code">delete</code> <code class="inline-code">run</code> 时进行 <code class="inline-code">overlap</code> 检查</li>
<li><code class="inline-code">add</code> 最多 8 个，大小固定 0x20，不限机会，但 <code class="inline-code">run</code> 之后不可用，必须先 <code class="inline-code">reload</code></li>
<li><code class="inline-code">delete</code> 功能释放所有堆块，最后一个槽位指针不置 0，两次机会</li>
<li><code class="inline-code">run</code> 可以显示堆块内容并释放 img chunk，两次机会</li>
<li><code class="inline-code">reload</code> 分配 img chunk，两次机会</li>
<li><code class="inline-code">rename</code> 分配 0x30，并可控 0x10，一次机会</li>
</ul>
